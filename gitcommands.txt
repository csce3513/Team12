GITHUB COMMANDS

ADDING FILES
------------
-Add a file to commit
$git add filename

-Create commit
$git commit -m 'commit name'

-Add remote location to commit to (if not already done)
$git remote add origin git@github.com:username/name.git

Push commit to remote location
$git push origin master




PULLING FILES
-------------
-Add remote location to keep track of updates (if not already done)
git remote add upstream git://github.com/username/name.git


-Pull in upstream changes
git fetch upstream
git merge upstream/master



BRANCHES
-------------
-Create branch
git branch branchname

-Switch between branches
git checkout branchname


-Merge with master
(while in master) git merge branchname
